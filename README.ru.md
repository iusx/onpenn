# onpenn

[English](README.md) · [中文](README.zh.md) · [Русский](README.ru.md)

Один комикс в день на языке, который вы понимаете. Каждый день показывается комикс с переведёнными текстовыми пузырями на китайском, английском и русском языках.

**Живой сайт:** развёрнут через GitHub Pages — смотрите ветку `gh-pages`.

---

## Внести перевод

### 1. Открыть визуальный редактор

Перейдите в браузере на `/zh/editor` (или `/en/editor`, `/ru/editor`) при запущенном локальном сервере разработки.

```bash
pnpm dev
# открыть http://localhost:3000/ru/editor
```

### 2. Расставить пузыри на изображении

- **Щелчок** в любом месте изображения создаёт новый пузырь в этой позиции.
- **Перетащите тело пузыря**, чтобы изменить его положение.
- **Перетащите правый край**, чтобы изменить ширину.
- **Перетащите нижний край**, чтобы изменить высоту.

### 3. Настроить каждый пузырь в боковой панели

| Поле | Описание |
|------|----------|
| Размер шрифта | Ползунок — значение в единицах `vw` (масштабируется с шириной изображения) |
| Шрифт | Выбор из доступных шрифтов (см. раздел [Пользовательские шрифты](#пользовательские-шрифты)) |
| Форма | `rect` (скруглённый прямоугольник) или `ellipse` (эллипс) |
| Фон | Выбор цвета + ползунок прозрачности |
| Позиция | `left` / `right` — персонаж, которому принадлежит пузырь |
| Текст | Переведённый текст для данного пузыря |

### 4. Скопировать вывод и вставить в `comics.ts`

Нажмите **Копировать** в боковой панели. Вывод выглядит так:

```ts
bubbles: [
  { id: 'b1', text: 'Ваш перевод здесь', position: 'left', x: 23.8, y: 18.4, w: 43.6, h: 16.8, fontSize: 1.5, bg: 'rgba(255,255,255,1)' },
]
```

Вставьте это в соответствующий блок перевода в `src/data/comics.ts`:

```ts
{
  id: "my-comic",
  image: "/comics/my-comic.jpg",
  author: "Автор",
  source: "Источник",
  sourceLanguage: "ru",          // язык текста, уже напечатанного в изображении — оверлеи скрываются для этой локали
  translations: {
    zh: {
      title: "漫画标题",
      fontFamily: "Yozai-Medium", // необязательно: шрифт по умолчанию для данного языка
      bubbles: [
        // ← вставить сюда
      ],
    },
    ru: {
      title: "Название комикса",
      bubbles: [
        // ← вставить сюда
      ],
    },
  },
}
```

### 5. `sourceLanguage`

Задайте `sourceLanguage` как локаль того языка, текст которого **уже напечатан в изображении**. Текстовые оверлеи автоматически скрываются при активации этой локали, чтобы оригинальный текст был виден без наложений.

## Пользовательские шрифты

Пользовательские файлы `.ttf` хранятся в `public/font/<locale>/`.

| Название семейства шрифтов | Файл | Язык |
|---------------------------|------|------|
| `Muyao-Softbrush` | `public/font/zh/Muyao-Softbrush.ttf` | Китайский (стиль кисти) |
| `Yozai-Medium` | `public/font/zh/Yozai-Medium.ttf` | Китайский (округлый) |

### Добавление нового шрифта

1. Скопируйте файл `.ttf` в `public/font/<locale>/YourFont.ttf`.
2. Объявите его в `src/app/globals.css`:

```css
@font-face {
  font-family: 'YourFont';
  src: url('/font/<locale>/YourFont.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```

3. Добавьте его в массив `FONTS` в `src/app/[locale]/editor/page.tsx`:

```ts
const FONTS = [
  { label: 'Отображаемое название шрифта', value: 'YourFont' },
  // ... существующие записи
]
```

4. Используйте название шрифта в полях `fontFamily` в `comics.ts` или выберите его в редакторе.

## Разработка

```bash
pnpm install
pnpm dev        # http://localhost:3000
pnpm build      # статический экспорт → out/
```

После пуша в ветку `main` автоматически выполняется деплой на GitHub Pages через workflow в `.github/workflows/deploy.yml`.
